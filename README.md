--Hệ thống lồng tiếng trực tiếp
Dự án này sử dụng model để nghe, dịch và lồng tiếng trực tiếp từ âm thanh hệ thống  sang tiếng Việt trong thời gian thực.

--Tính năng chính 
Real-time Speech-to-Text: Sử dụng mô hình faster-whisper để chuyển đổi giọng nói tiếng Anh thành văn bản.

Neural Machine Translation: Dịch thuật chất lượng cao từ tiếng Anh sang tiếng Việt bằng mô hình NLLB-200.

Vietnamese TTS: Lồng tiếng Việt tự nhiên thông qua bộ máy Valtec TTS.

System Audio Capture: Thu âm trực tiếp từ máy tính thông qua VB-Audio Virtual Cable.

--Yêu cầu hệ thống 
OS: Windows 10/11.

GPU: NVIDIA RTX (recomment ít nhất GTX 5060 vì khá nặng nếu ít hơn sẽ có hiện tượng dịch nhảy câu và không nói hết ).

Software:

Python 3.10+.

VB-Audio Virtual Cable (Để dẫn hướng âm thanh từ Chrome vào Python).
